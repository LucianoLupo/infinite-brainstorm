# P3.4: HiDPI + minimap + PNG export + snap-to-grid

XL item, sequenced last. Land as much as possible green; multi-board deferred.

## Steps

1. **HiDPI (devicePixelRatio)** — backing store = display * dpr; `ctx.set_transform(dpr,..)`;
   `render_board` clears/grids using CSS dimensions (display size), not `canvas.width()`.
   - add `dpr: f64` to `RenderState`; clamp 0.5..=4.0.
   - verify: crisp on Retina; box-select/hit-test still correct (those use CSS coords already).

2. **Minimap** — small overlay canvas drawing scaled node rects + viewport rect; click-to-recenter.
   - pure helper `minimap_transform(bbox, mw, mh) -> (scale, off_x, off_y)` in app.rs + tests.
   - reuse render Effect (track board + camera); recenter on click.

3. **PNG export** — `canvas.to_data_url("image/png")` reusing the download-anchor pattern.
   Viewport export. Add an "Export PNG" affordance (toolbar; always visible).

4. **Snap-to-grid on drag release** — round moved nodes' x/y to nearest 50px.
   - pure helper `snap_to_grid(v, grid)` + tests. Applied in on_mouse_up when a drag committed.

5. **Multi-board** — DEFER (lowest priority).
